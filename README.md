# PhotoWhisperer

Photography settings calculator. See `arch-spec-v3.1.md` for architecture and `screen-spec-v1.md` for UI/UX spec.

## Stack
Next.js 14+ App Router · TypeScript · Tailwind v4 · Supabase · Stripe · Anthropic Sonnet

## Development
```bash
pnpm install
pnpm dev
```

Open http://localhost:3000
